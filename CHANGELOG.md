## Migration note

This repository is the clean public successor to a private predecessor that mixed UGV-platform work with earlier work on a different platform. The git history starts fresh here; the predecessor's history is preserved privately for archaeological reference but is not relevant to this codebase's identity.

Subsystem commits arrive incrementally as the migration completes. Each subsystem will land as a single atomic commit with full reasoning in the commit body, so the early git history reads as a deliberate architectural assembly rather than as the messier reality of how the code was originally written.
